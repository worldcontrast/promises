Operational rules — hardcoded into agent system, non-negotiable
Zero Contact Policy
Agents never submit forms, login, or interact with any site
No emails, no DMs, no API keys from candidates
No partnerships with campaigns or parties
All data collection is passive read-only observation
Promises Only Doctrine
Extractor agent configured: collect commitments, reject attacks
No negative framing, no comparison of candidates by agents
Verbatim quotes only — no paraphrasing of political content
Ambiguous statements flagged, never silently interpreted
Symmetric Coverage Rule
All official candidates in an election receive identical coverage
Same 9 categories applied to every candidate without exception
If data is unavailable for one, all show the same "no data" state
No candidate receives more or fewer collection attempts
